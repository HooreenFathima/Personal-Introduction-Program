# Personal-Introduction-Program
Week 1 : Developers Arena Internship Program 
-----------------------------------------------------------PROJECT OVERVIEW -------------------------------------------------------
This is a Python language interactive program written as part of my internship at **The Developers Arena**. The program takes  user details such as name, age, and hobbies as input to generate a personalized and friendly welcome message.

----Running Platform : VS Code

 -------------------How to Run the Program----------------------
Follow these steps to run the project:

1. Ensure you have Python 3.10 installed on your system if not install .
2. **Clone the Repository**: Download these files or clone the repo using:
   ```bash
   git clone <your-repository-link>
   ```
3.Open in VS code , load the files 
4.Run the File 
5.Enter the Details when code executes and prompts shown on screen .
